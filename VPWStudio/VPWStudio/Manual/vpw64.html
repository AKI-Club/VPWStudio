General Notes

Work on Virtual Pro-Wrestling 64 likely started after the development of Virtual Pro-Wrestling on the PS1. It's unknown if it was started after WCW vs. the World, but based on early VPW64 data found in WCW vs. nWo: World Tour, the game did not originally feature any licensed wrestlers.

The game's working/original title was Virtual Pro-Wrestling: Ultra Battle Royal (バーチャル・プロレスリング　ウルトラバトルロイヤル).

There are some unused and/or unfinished characters hidden in the game data.

Most game text is in EUC-JP encoding.

Wrestler Data

Z64 ROM address: 0x49438

The wrestler list consists of many pointers to the wrestler definitions. There are more entries than one would expect, due to a number of entries being duplicated for the cruiserweight-only roster.

Profile

Profile data is stored in the ROM. This includes wrestler names, height/weight, and various other items.

All names are using EUC-JP encoding.

  • Long Name (max length 16 characters)
  • Short Name (max length 8 characters)

VPW Studio does not currently allow you to edit wrestler names, since VPW64 stores them in a way that is not easy to work with.

Moves and Parameters

Moves and Parameters are found in the FileTable, but the data formats are not fully understood yet.

Default Costumes

Stable Data

Z64 ROM address: 0x4989C

Each stable entry contains multiple values:

  • Offset 0x00: Pointer to wrestler definitions for this stable (4 bytes)
  • Offset 0x04: Number of wrestlers in stable (4 bytes)
  • Offset 0x08: Pointer to championship definitions for this stable (4 bytes)
  • Offset 0x0C: Number of championships in stable (4 bytes)
