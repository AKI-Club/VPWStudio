General Notes

World Tour's codebase is based on an earlier version of Virtual Pro-Wrestling 64. This is supported by various leftover data found in the World Tour ROM (even in v1.1/Rev A).

"Sting"/fake Sting/nWo Sting, Glacier, and Wrath are not in Virtual Pro-Wrestling 64.

WCW vs. nWo World Tour is the only AKI N64 game that requires the Controller Pak for saving game data (including unlocked wrestlers). You may want to take advantage of an "Unlock All Wrestlers and Championships" GameShark code to make your life easier:

Stable Data

Game Variant → NTSC-U v1.0 NTSC-U v1.1 PAL
Z64 ROM address 0x37EC8 0x37F38 0x37EC8

Each stable entry contains multiple values:

  • Offset 0x00: Pointer to wrestler definitions for this stable (4 bytes)
  • Offset 0x04: Number of wrestlers in stable (4 bytes)
  • Offset 0x08: Pointer to championship definitions for this stable (4 bytes)
  • Offset 0x0C: Number of championships in stable (4 bytes)
