General Notes

WCW/nWo Revenge is a considerable upgrade from World Tour and VPW64, and forms the basis of the rest of AKI's N64 wrestling game library.

A number of wrestlers were planned to be in the game, based on reports from AKI, as well as gaps in the wrestler ID4 numbering. Ric Flair and the Ultimate Warrior are the only confirmed characters who did not find their way into the game.

Wrath is an unfinished wrestler, and can normally only be unlocked using a GameShark code. There are some small problems with some of his moves, but he is otherwise playable without issue.

Notable File Types

DoubleTex

As the name may suggest, these files contain data for two AkiTextures, stored back to back. These are only used for the ending/credits sequence.

Stable Data

Game Variant → NTSC-U PAL
Z64 ROM address 0x3B0F8 0x38548

Stables in WCW/nWo Revenge hold up to 9 wrestlers maximum.

Each Stable entry consists of three values:

  • Offset 0x00: Pointer to wrestler definitions for this stable (4 bytes; "Wrestler Definitions" in Stable Definitions dialog)
  • Offset 0x04: Number of wrestlers in stable (2 bytes; "Wrestler Count" in Stable Definitions dialog)
  • Offset 0x06: Stable header graphic File ID (2 bytes; "Header Graphic" in Stable Definitions dialog)

Costume Colors

Costume colors in Edit Mode are defined in file ID 0A51.

The default values from the NTSC-U version of WCW/nWo Revenge are displayed in the table below. Each table cell contains three values.
The first line represents the hex value of the color as it appears in the palette file.
The second line represents the amount of Red, Green, and Blue in the range 0-31.
The third line represents the amount of Red, Green, and Blue in the range 0-255, as interpreted by VPW Studio (circa pre-Alpha Preview 18). These values are used for the background color of each cell.
